📌 Exploring and Visualizing Iris Dataset

🎯 Objective 

To load, explore, and visualize the Iris dataset to understand data distribution, feature relationships, and detect patterns.

📊 Dataset Used

•	Iris Dataset (built-in from Seaborn / Scikit-learn)

•	Contains measurements of 3 species of flowers:

      o	Setosa
  
      o	Versicolor
   
      o	Virginica

✏️ Features

•	Sepal Length

•	Sepal Width

•	Petal Length

•	Petal Width

🧠 Tools & Libraries Used

•	Python

•	Pandas

•	Matplotlib

•	Seaborn

📈 Visualizations Performed

✔ Scatter Plot (feature relationships)

✔ Histograms (data distribution)

✔ Box Plots (outlier detection)

🔍 Key Results & Findings

•	Petal length and petal width are highly useful in distinguishing species

•	Setosa is clearly separable from other species

•	Some overlapping exists between Versicolor and Virginica

•	No major missing values in dataset

🚀 Conclusion

This task helped in understanding basic data exploration and visualization techniques used in machine learning preprocessing.
